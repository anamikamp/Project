# 🎯 Number Guessing Game

A command-line Number Guessing Game built with Python as part of the **roadmap.sh Number Guessing Game** project.

## 📌 Project Page

https://roadmap.sh/projects/number-guessing-game

## 🚀 Features

* Three difficulty levels:

  * Easy (10 chances)
  * Medium (5 chances)
  * Hard (3 chances)
* Random number generation between 1 and 100
* Hint system for Easy and Medium difficulty levels
* Input validation for user guesses
* Displays whether the guessed number is higher or lower than the target
* Tracks the number of attempts
* Measures the time taken to complete the game
* Play Again option
* Displays the correct number when the player loses

## 🛠️ Technologies Used

* Python 3
* `random` module
* `time` module

## 📋 Requirements

* Python 3.x installed on your system

## ▶️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/anamikamp/Project.git
```

2. Navigate to the project folder:

```bash
cd <Project>
```

3. Run the game:

```bash
python number_guessing_game.py
```

## 🎮 How to Play

1. Choose a difficulty level.
2. Guess a number between **1** and **100**.
3. Follow the hints provided during the game.
4. Guess the correct number before running out of chances.
5. Choose whether to play another round.
